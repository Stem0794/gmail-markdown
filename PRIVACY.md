# Règles de confidentialité pour Gmail Markdown

## 1. Introduction
Ce document décrit comment l'extension Chrome "Gmail Markdown" gère vos données personnelles.

## 2. Collecte de données
L'extension ne collecte aucune donnée personnelle en dehors de votre environnement local. Aucune information n'est envoyée à des serveurs tiers.

## 3. Utilisation des données
Les données traitées par l'extension sont uniquement destinées à la conversion des emails en format Markdown.

## 4. Sécurité
Toutes les opérations de conversion sont effectuées localement dans votre navigateur.

## 5. Contact
Pour toute question, veuillez consulter le dépôt GitHub du projet.
